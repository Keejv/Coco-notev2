## To install
```git clone https://github.com/Keejv/Coco-notev2.git
cd Coco-notev2
npm / yarn install
```

## Server
```npm / yarn run server
Local server will start at http://localhost:5000
```

## Client
```cd Coco-notev2/client
npm / yarn run start
Local server will start at http://localhost:3000
```
